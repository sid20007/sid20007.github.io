# Siddhartha G — Portfolio

Personal portfolio built with **Next.js 16**, **React 19**, and **Tailwind CSS 4**.

## Main Focus
Building premium, high-performance web experiences. This project serves as a showcase of my technical stack and design philosophy.

## Tech Stack
- **Framework:** Next.js 16.2 (App Router)
- **Library:** React 19.2
- **Styling:** Tailwind CSS 4
- **Animations:** Framer Motion 12
- **Font:** Geist Sans & Mono

## Development
```bash
npm run dev
```
